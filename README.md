# Usage

**Recommended OS**: Ubuntu 20.04 LTS

**Requirements**: Docker, Docker Compose, Yarn, Node.js

**Development**

- `make dev` (Start development backend services, http://localhost:3000)
- `make test` (Run integration tests)
